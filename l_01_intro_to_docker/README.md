# Lab I: Introduction to docker


### Requirements
- bash
- docker


### Setup
-   ```bash
    git clone https://github.com/mak9su4roi/big-data-2022
    ```
-   ```bash
    cd l_01_intro_to_docker
    ```
-   ```bash
    chmod +x build.sh
    ```

### Build
-   ```bash
    ./build.sh lab_01
    ```

### Run
-   ```bash
    docker run lab_01
    ```
